# TeamPulse ⚡

Modern full-stack task management dashboard built using React and Flask.

## 🚀 Features

- Secure JWT Authentication
- Drag and Drop Kanban Board
- Task Status Management
- Priority Levels
- Due Dates
- Modern Dark UI
- Responsive Dashboard
- Full CRUD Operations
- Real-time Task Updates

---

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript
- CSS
- React Beautiful DnD

### Backend
- Flask
- Flask JWT Extended
- SQLAlchemy
- SQLite
